## 🐛 常见问题

### Q1: 星球不旋转？
**A:** 检查浏览器性能，确保支持Canvas 2D。尝试刷新页面。

### Q2: 卫星点击没反应？
**A:** 确保鼠标悬停时卫星有放大效果。点击时会有虫洞跃迁动画。

### Q3: 虫洞跃迁卡顿？
**A:** 正常情况1.8秒流畅过渡。如果卡顿，可能是设备性能问题。

### Q4: 登录后没有主控台？
**A:** 主控台会显示3秒后自动进入应用。如果没看到，检查控制台报错。

### Q5: 稻田不生长（农户）？
**A:** 需要在注册时填写邀请码才会触发动画。

### Q6: 购物车没有冒光（买家）？
**A:** 购物车数量≥3时才会冒光。当前演示固定为5件。

### Q7: 火箭不升空（专家）？
**A:** 火箭在主控台会上下浮动，满100%时会全屏升空（演示版未触发）。

## 📊 性能检查

### 浏览器开发者工具
1. 打开F12 → Performance
2. 录制星球界面3秒
3. 检查FPS（应≥50fps）
4. 检查内存占用（应<100MB）

### 网络面板
1. F12 → Network
2. 刷新页面
3. 检查首屏JS大小（应≤200KB）
4. 检查加载时间（应≤2s）
