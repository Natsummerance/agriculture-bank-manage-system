## 🚀 三步操作流程

### G1 - 接收实时消息
```
1. 点击 → 右上角铃铛图标
2. 查看 → 未读消息列表
3. 点击 → 消息标记已读
```

### G2 - 查询区块链存证
```
1. 进入 → 金融页或交易页
2. 点击 → 「区块链存证」按钮
3. 搜索 → 输入哈希或标题
4. 查看 → 点击记录查看详情
```

### G3 - 发起联合贷款
```
1. 进入 → 金融页
2. 滚动 → 到「高级功能」区域
3. 点击 → 「多人联合贷」卡片
4. 操作 → 邀请银行、分配份额
5. 生成 → 电子联名合同
```

### D1 - 查看日夜轨道
```
1. 进入 → 首页
2. 查看 → 顶部3D星球
3. 观察 → 自动色温变化
4. 测试 → 右下角手动切换按钮
```

### D2 - 查看额度热力图
```
1. 进入 → 金融页
2. 滚动 → 到「高级功能」区域
3. 点击 → 「全国额度热力图」卡片
4. 交互 → 鼠标悬停查看省份
5. 观察 → 热力脉冲和统计数据
```

## 🎯 常见问题速查

### Q: 找不到消息中心？
**A:** 看右上角，铃铛图标旁边的圆形按钮，任何页面都有。

### Q: 金融页没看到SP1功能？
**A:** 向下滚动到页面底部，找到「高级功能」标题。

### Q: 区块链存证在哪里？
**A:** 两个地方：
  1. 金融页底部「高级功能」第2张卡片
  2. 交易页左下角「订单存证」按钮

### Q: 热力图不旋转？
**A:** 检查是否打开了全屏模式，鼠标移动会影响旋转。

### Q: 星球颜色不变？
**A:** D1星球根据系统时间自动切换（6:00-18:00为日间）。
      可以点击右下角按钮手动切换测试。
