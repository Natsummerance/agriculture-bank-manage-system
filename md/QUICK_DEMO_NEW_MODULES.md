# 🚀 新增模块快速演示指南

> 5分钟快速体验所有新功能

## 🎯 演示流程

### 第一步：启动应用
```bash
npm install
npm run dev
```

访问 `http://localhost:5173`

---

### 第二步：进入应用主界面

1. **选择星球** → 点击「买家星球」（蓝色）
2. **登录** → 输入手机号 + 验证码
3. **等待仪表盘动画** → 3秒后自动进入主界面

---

## 📦 模块一：购物车完整体验

### 1. 打开购物车
- 点击顶部导航栏**购物车图标** 🛒
- 查看气泡数字（5件商品）

### 2. 管理商品
```
✅ 点击全选 → 所有商品被选中
✅ 单击商品复选框 → 取消单个商品
✅ 点击 [-] 按钮 → 数量减1
✅ 点击 [+] 按钮 → 数量加1
✅ 手动输入数量 → 自动校验范围
✅ 点击删除图标 → 弹出确认对话框
```

### 3. 结算体验
```
观察底部悬浮栏：
- 已选商品数量：2件
- 合计金额：实时更新
- 点击「去结算」→ 跳转订单确认页
```

### 4. 空态体验
- 删除所有商品 → 显示空态插画
- 点击「去逛逛」→ 返回首页

**预计耗时**: 2分钟

---

## 🔔 模块二：消息通知系统

### 1. 打开消息抽屉
- 点击顶部导航栏**铃铛图标** 🔔
- 观察未读红点脉冲动画（3条未读）
- 抽屉从右侧滑入

### 2. 筛选消息
```
点击Tab标签：
- 「全部」→ 显示所有消息
- 「系统」→ 只显示系统通知
- 「订单」→ 只显示订单消息
- 「消息」→ 只显示IM消息
```

### 3. 消息交互
```
✅ 点击任意消息 → 标记为已读 + 跳转详情
✅ 悬停消息 → 显示删除按钮
✅ 点击删除 → 消息消失（动画退出）
✅ 点击「全部已读」→ 红点消失
```

### 4. 关闭抽屉
- 点击遮罩层 or 右上角 ✕ 按钮

**预计耗时**: 1分钟

---

## 💬 模块三：专家咨询IM

### 1. 打开咨询浮窗
方式一：点击任意商品详情页「咨询专家」按钮  
方式二：专家列表页「立即咨询」

### 2. 发送消息
```
文字消息：
- 输入框输入文字 → 点击发送 ✈️
- 按下 Enter 键 → 快速发送

语音消息：
- 按住麦克风图标 🎤
- 观察录音时长（最多60秒）
- 松开发送 / 上滑取消

快捷短语：
- 点击预设短语 → 自动填充到输入框
```

### 3. 高级功能
```
✅ 点击电话图标 📞 → 语音通话
✅ 点击视频图标 📹 → 视频咨询
✅ 点击「预约专家」→ 跳转日历页面
✅ 上传图片/文件 → 展示缩略图
```

### 4. 在线状态
- 观察专家头像右下角绿点（在线）

**预计耗时**: 2分钟

---

## 🛍️ 模块四：产品详情页

### 1. 进入详情页
- 首页瀑布流 → 点击任意商品卡片

### 2. 图片浏览
```
✅ 左右滑动/点击圆点 → 切换图片
✅ 点击「360°全景」标签 → 播放视频
✅ 点击分享图标 → 复制链接
✅ 点击收藏图标 ❤️ → 变红填充
```

### 3. 查看详情
```
切换Tab标签：
- 「详情」→ 图文介绍
- 「参数」→ 规格表格
- 「评价」→ 用户评价瀑布流
```

### 4. 底部操作
```
观察底部双按钮布局（间距12px）：

左侧：「加入购物车」
- 点击 → 飞入动画（360°旋转）
- Toast提示 → 购物车数量+1

右侧：「立即购买」
- 点击 → 直达订单确认页
```

**预计耗时**: 2分钟

---

## 📝 模块五：发布求购需求

### 1. 进入发布页
- 买家首页顶部 → 点击「发布需求」按钮（绿色）

### 2. 填写表单
```
智能搜索：
- 输入「苹」→ 自动联想「有机富硒苹果」
- 点击建议 → 自动填充

数量 & 单位：
- 输入「500」
- 选择单位「kg」

价格期望：
- 拖动滑块 → 实时显示价格
- 滑到最左侧 = 面议

交货日期：
- 点击日历图标 📅
- 选择未来日期（过去日期禁用）

交货地址：
- 输入详细地址（可集成高德地图）

附件图片：
- 点击上传区 → 模拟上传（最多3张）
- 悬停图片 → 显示删除按钮
```

### 3. AI预填充体验
```
点击右上角「AI预填充」按钮：
→ 弹出AI推荐方案卡片
→ 预览内容
→ 点击「应用此方案」→ 表单自动填充
```

### 4. 发布需求
```
✅ 表单校验通过
✅ 粒子动画效果（30个白色粒子上浮）
✅ 点击「发布需求」
✅ Toast: 正在智能匹配农户...
✅ 1.5秒后跳转需求列表
```

**预计耗时**: 3分钟

## 🎁 附加功能演示

### 发票下载（管理后台）
```
功能位置：订单详情页 → 点击「下载发票」

体验流程：
1. 预览完整电子发票（PDF格式）
2. 点击「下载PDF」→ 保存到本地
3. 点击「发送邮箱」→ 输入邮箱地址
4. 观察区块链存证验证标识 ✅
```

### 管理员推送通知
```
功能位置：管理后台 → /admin/push

体验流程：
1. 输入推送标题（50字内）
2. 输入推送内容（200字内）
3. 上传推送图片（可选）
4. 选择目标用户群体：
   - 勾选「农户」→ 显示12,543位用户
   - 勾选「买家」→ 显示8,932位用户
5. 定时发送（可选）：
   - 选择未来时间 → 自动定时
6. 实时预览推送效果
7. 点击「立即发送」→ Toast提示发送成功
```

### 灰度发布控制台
```
功能位置：管理后台 → /admin/feature-flag

体验流程：
1. 查看功能列表（AI匹配v2.0、区块链v2.0等）
2. 打开功能开关 → Switch动画
3. 拖动流量滑块：
   - 0% → 无用户看到新功能
   - 50% → 一半用户看到新功能
   - 100% → 全量发布
4. 观察实时监控图表：
   - 活跃用户数（蓝线）
   - 错误数（红线）
5. 快速操作：
   - 点击「紧急回滚」→ 流量归零
   - 点击「全量发布」→ 流量100%
```
