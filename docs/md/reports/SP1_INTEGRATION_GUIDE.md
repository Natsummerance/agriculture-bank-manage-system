# 🎯 SP1功能集成指南

## 功能分布总览

SP1阶段的所有功能已经无缝集成到各个页面中，用户可以在实际使用场景中访问这些功能。

---

## 📍 功能位置地图

### 🏠 首页（HomePage）

#### **D1 - 日夜轨道星球** ✨
- **位置**：页面顶部Hero区域
- **功能**：
  - 自动日夜切换（6:00-18:00为日间）
  - 恒星光源色温变化（6500K→3000K）
  - 夜间卫星冷光轨迹
  - 荧光尘埃粒子动画
- **交互**：
  - 鼠标移动：星球视差效果
  - 右下角按钮：手动切换日夜模式
  - 左下角：实时显示色温和粒子数

---

### 💰 金融页面（FinancePage）

#### **G3 - 多人联合贷** 🏦
- **位置**：页面底部「高级功能」区域，第一张卡片
- **标识**：黄色图标 + G3标签
- **功能**：
  - 银行发起联合贷项目
  - 邀请同行参与
  - 份额分配管理
  - 电子联名合同生成
- **入口**：点击卡片打开完整界面

#### **G2 - 区块链存证** 🔒
- **位置**：页面底部「高级功能」区域，第二张卡片
- **标识**：绿色图标 + G2标签
- **功能**：
  - 合同/订单/还款存证查询
  - Polygon zkEVM区块链浏览
  - 零Gas费用
  - 司法级存证
- **入口**：点击卡片打开区块链浏览器

#### **D2 - 全国额度热力图** 🗺️
- **位置**：页面底部「高级功能」区域，第三张卡片
- **标识**：青色图标 + D2标签
- **功能**：
  - 20省份可贷额度可视化
  - 实时热力映射（绿→金→橙→红）
  - 5分钟自动更新
  - 鼠标悬停省份详情
- **入口**：点击卡片打开全屏热力图

---

### 🛒 交易页面（TradePage）

#### **G2 - 订单存证** 📦
- **位置**：页面左下角，「发布求购需求」按钮下方
- **标识**：玻璃态按钮 + 锁图标 + G2标签
- **功能**：
  - 订单区块链追溯
  - 交易哈希查询
  - 物流存证
- **入口**：点击「订单存证」按钮

---

### 🔔 全局功能

#### **G1 - 统一消息中心** 💬
- **位置**：全局右上角固定按钮（铃铛图标）
- **特性**：在任何页面都可访问
- **功能**：
  - WebSocket实时推送
  - 5类消息（审批/预约/系统/支付/专家）
  - 离线Push通知
  - 通知设置（持久化）
  - 心跳检测 + 断线重连
- **入口**：点击右上角铃铛图标

## 🚀 使用流程示例

### 场景1：申请联合贷款
1. 导航到「金融」页面
2. 滚动到「高级功能」区域
3. 点击「多人联合贷」卡片（G3）
4. 查看项目信息
5. 点击「邀请参与行」
6. 选择银行并设置份额
7. 发送邀请
8. 生成电子联名合同

### 场景2：查询合同存证
1. 导航到「金融」页面
2. 滚动到「高级功能」区域
3. 点击「区块链存证」卡片（G2）
4. 使用搜索框或筛选按钮
5. 点击存证记录查看详情
6. 复制哈希或在浏览器中打开

### 场景3：查看额度分布
1. 导航到「金融」页面
2. 滚动到「高级功能」区域
3. 点击「全国额度热力图」卡片（D2）
4. 鼠标悬停在省份上查看详情
5. 查看右下角统计数据
6. 观察自动旋转和热力脉冲

### 场景4：接收实时消息
1. 在任何页面
2. 观察右上角铃铛图标的未读计数
3. 点击铃铛打开消息中心
4. 查看WebSocket连接状态（绿点 = 已连接）
5. 点击消息标记为已读
6. 点击齿轮设置通知偏好
