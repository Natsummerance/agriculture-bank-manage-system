## 🐛 故障排查

### 消息中心不显示？
- 确保页面已完全加载
- 刷新页面重新连接WebSocket
- 检查浏览器控制台是否有错误

### 动画卡顿？
- 关闭其他占用资源的标签页
- 检查GPU硬件加速是否开启
- 降低浏览器缩放比例

### 通知权限被拒绝？
- 在浏览器设置中允许通知权限
- 刷新页面重新请求权限
